파이썬에서 문자끼리 순서를 비교할 경우나,

"c" 부터 시작하는 5개 알파벳을 구하라. 라는 문제를 풀때 사용할수있다.

아래와 같이 외워서 써먹자.


* 문자의 순서 ORDER -> ord함수 하용

* 숫자를 문자로 CHARACTER -> chr함수 사용

```python
a_ascii = ord('a')
a_str = chr(a_ascii)
```

"c" 부터 시작하는 5개 알파벳을 구하라.

```python
for i in range(5):
  print(chr(ord("c") + i))
```
